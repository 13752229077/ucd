So, what is this?

Nosejob is a code sniffing software prototype, which helps the client to receive user-friendly analytics on the quality of their project.

How to use?

  1. In the Home section, press the Open Project button and locate your project's directory. You will be redirected to the Overview section.
  2. Overview section provides general analytics of the project. For some specifics, hover over bars to see the values of the severity/occurrence charts.
  3. If more curious, the Details section provides specific analysis per file for supported smells.

Supported smells

  • Long Method
  • Long Class
  • Cyclomatic Complexity
  • Data Clumps
  • Duplicated Code
  • Feature Envy
  • Heavy Commenting
  • Lazy Class
  • Refused Bequest
Head over to the other tabs to see what these smells mean, and how to refactor them!
