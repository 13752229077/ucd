What is a Long Method smell?

A method contains too many lines of code.

How can we treat a long method?

  • An extract method can be used to reduce the size of a method.
  • If local variables and parameters interfere with extracting a method, use Replace Temp with Query, Introduce Parameter Object or Preserve Whole Object.
  • We can move the entire method into a separate object.
  • If methods have a lot of conditional loops and operators then code can more than likely be moved to a different method, we can use Decompose Conditional to do this.

Why bother?

  • The shorter and more concise a method is the easier it is to maintain,understand and work on in the future.
  • Duplicated code is more often than not found in long methods.
